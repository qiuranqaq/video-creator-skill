# Reference: 2D Animation & Motion Graphics
**适用场景**: 无对白动画、带旁白的 MG 动画、扁平风格宣传片、解释类视频。

## 1. 叙事双模 (Dual Narrative Modes)

### A. 纯视觉模式 (Silent Storytelling)
*   **肢体语言**: 利用夸张的形变 (Squash & Stretch) 表现情绪。
*   **画面隐喻**: 用图形的变换代替文字说明（如：破碎的爱心转场变成裂开的大地）。

### B. 旁白配合模式 (Narrated Storytelling)
*   **音画同步 (Syncing)**: 画面必须领先旁白 0.5s 出现，引导观众预期。
*   **关键词跳出 (Keyword Reveal)**: 旁白说到核心动词或名词时，画面同步出现动态文字或对应图标，加强记忆。
*   **呼吸感**: 旁白停顿时，画面应有大幅度的动作或转场；旁白密集时，画面保持相对平稳，避免信息过载。

## 2. 现代排版与文字融合 (Modern Text Integration)
拒绝过时的“抖动/变色”式特效，文字应当具有**建筑感**和**物理属性**：

*   **Masking Reveal (遮罩显现)**: 文字不是飘进来的，而是被物体“刷”出来的。例如：车驶过，车身后显现出标语；或者文字从一条极细的线条中“生长”出来。
*   **Diegetic Text (场景内嵌)**: 文字作为环境的一部分存在。例如：它是墙上的投影、地面的阴影、或者主角衣服上的图案。
*   **Swiss Grid (瑞士网格)**: 强调严谨的排版构图。文字本身就是几何图形，通过字重 (Weight) 和间距 (Kerning) 的丝滑变化来匹配音乐节奏。
*   **UI/HUD Elements**: 在科技感题材中，文字作为悬浮的 UI 界面出现，伴随细微的数据滚动和光标闪烁。

## 3. 视觉节奏 (Visual Rhythm)
*   **Anticipation (预备动作)**: 动作发生前的反向蓄力。
*   **Transitions (转场)**: 
    *   **Shape Match**: 利用相同形状无缝衔接。
    *   **Wipe**: 快速划过屏幕带出新场景。
