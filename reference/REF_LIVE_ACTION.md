# Reference: Live Action & TVC Production
**适用场景**: 品牌 TVC、剧情短片、宣传片、访谈。
**核心逻辑**: **Execution (执行力)** + **Cinematography (电影感)**。

## 1. 制作流程标准 (Standard Workflow)

### Phase 1: Pre-Production (筹备期)
*   **PPM (Pre-Production Meeting)**: 确认演员(Casting)、服装(Styling)、场地(Location)、道具(Props)。
*   **Recce (勘景)**: 确认现场光照条件、电源位置、噪音干扰。
*   **Shot List (拍摄清单)**: 决定拍摄顺序（通常不按剧情顺序，而是按场地/灯光顺序拍）。

### Phase 2: Production (拍摄期)
*   **Blocking (走位)**: 导演、摄影指导、演员确认动线。
*   **Lighting (布光)**: 铺设主光(Key)、辅光(Fill)、轮廓光(Rim)。
*   **Shooting (实拍)**: 每一个镜头需记录 Take 数，标记 Good Take。

## 2. 器材与配置分级 (Equipment Tiers)

根据预算规模，选择对应的设备方案：

### Level A: 电影级/TVC (Cinema Grade)
*   **Camera**: ARRI Alexa Mini LF / RED V-Raptor (电影机)。
*   **Lenses**: Cooke S4 / Zeiss Master Primes (定焦电影头组)。
*   **Support**: Fisher Dolly (移动摄影车) / Technocrane (大摇臂) / Steadicam。
*   **Lighting**: 
    *   ARRI M-Series (M18/M40) 镝灯（模拟阳光）。
    *   Skypanel S60/S360（大面积柔光）。
    *   蝴蝶布/米菠萝（控光附件）。
*   **Crew**: 导演、制片、DP、灯光师(Gaffer)+大助/二助、美术指导、收音师、DIT。

### Level B: 商业短片/宣传片 (Commercial)
*   **Camera**: Sony FX6 / FX9 / Canon C70。
*   **Lenses**: Sony G Master / Sigma Art (变焦为主，定焦为辅)。
*   **Support**: Easyrig (易事背) / Ronin RS3 Pro (手持云台) / 滑轨。
*   **Lighting**: 
    *   Aputure (爱图仕) 600d/300x LED灯组。
    *   Nanlite PavoTube (灯管) 补氛围。
*   **Crew**: 导演兼摄影、灯光助理、化妆师、制片助理。

### Level C: 轻量化/内容电商 (Run & Gun)
*   **Camera**: Sony A7S3 / Fuji X-H2s / iPhone 15 Pro Max。
*   **Lenses**: 24-70mm F2.8 标准变焦头。
*   **Support**: 手持 / 三脚架。
*   **Lighting**: 神牛 LED 灯 / 柔光箱 / 自然光反光板。
*   **Crew**: 摄影师（单兵作战）+ 编导。

## 3. 现场执行术语 (On-Set Terminology)
*   **Coverage (覆盖镜头)**: 除了主镜头，多拍几个特写和反应镜头以防剪辑接不上。
*   **Room Tone (环境音)**: 拍完后全场安静 30秒录制环境底噪。
*   **Golden Hour (黄金时刻)**: 日出后/日落前 1小时，光线最柔和。
*   **Flag (遮光旗)**: 用来遮挡多余光线，控制对比度。
