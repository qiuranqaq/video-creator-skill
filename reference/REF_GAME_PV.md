# Reference: Game PV & Trailer Production
**适用场景**: 游戏版本前瞻、新角色演示、活动预告、世界观剧情 PV。
**核心逻辑**: **Audio-Visual Sync (音画同步)** + **Hype Creation (情绪煽动)**。

## 1. 四大核心类型 (The 4 Archetypes)

### A. 角色演示 (Character Demo / Trailer)
*   **结构**: 
    1.  **Intro**: 角色立绘/模型特写 + 标志性台词 (Entrance Quote)。
    2.  **Kit Showcase**: 普攻 (Normal Attack) -> 小技能 (Skill) -> 被动展示 (Passive)。
    3.  **Fluff**: 待机动作、回城动画、趣味交互 (展示性格)。
    4.  **Climax**: 终极技能 (Ultimate) 配合处刑曲高潮。
    5.  **Outro**: 结算 Pose + 角色名/卡池时间。
*   **关键技**: 技能释放必须**踩点 (On Beat)**。

### B. 版本/大型活动 PV (Version/Event Trailer)
*   **结构**:
    1.  **Tease**: 新地图/新场景的空镜 (Fly-through)。
    2.  **Threat**: 新 Boss 或敌人的压迫感展示。
    3.  **Solution**: 新角色/新皮肤 (Skins) 混剪展示。
    4.  **Feature**: 新玩法机制 (Minigames/Mechanics) 快速混剪。
    5.  **CTA**: 版本更新日期 + 登录送福利。

### C. 剧情先导 (Story/Lore Teaser)
*   **风格**: 谜语人 (Cryptic)、碎片化叙事、黑深残或史诗感。
*   **技法**: 
    *   **Voiceover (旁白)**: 只有声音，画面留白或黑屏。
    *   **Flashback**: 快速闪回，黑白画面与彩色画面交替。
    *   **Symbolism**: 燃烧的旗帜、破碎的镜子等隐喻符号。

### D. 玩法实机 (Gameplay Trailer)
*   **重点**: UI 界面展示（可选）、真实的伤害数值、连招流畅度。
*   **红线**: 既然是实机，尽量减少 CG 欺诈，用剪辑节奏弥补画面不足。

## 2. 游戏 PV 专用术语 (Game PV Lexicon)

*   **Hit Stop (顿帧/卡肉)**: 攻击命中瞬间画面停顿 0.1s，增强打击感。
*   **Screen Shake (屏幕震动)**: 配合重击或爆炸的摄像机震动。
*   **Speed Ramping (变速/时间重映射)**: 动作前慢动作 (Anticipation) -> 攻击瞬间极快 (Impact) -> 命中后慢动作 (Follow through)。
*   **Match Cut (匹配剪辑)**: 不同角色的同一个动作（如挥剑）无缝衔接。
*   **Drop (高潮点)**: 音乐副歌响起瞬间，必须配合最华丽的画面（大招/爆炸/集结）。
